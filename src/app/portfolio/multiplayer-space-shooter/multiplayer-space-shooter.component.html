In this game, players spawn on a randomly generated map. Each player can control their spaceship's engines using arrow keys, ship's direction with mouse and fire missiles by pressing space bar. The goal is to destroy every other spaceship in the game.

The game is made with in C++ using SFML library. Multiplayer implementation doesn't support local client prediction. The code is available on https://github.com/Ilethas/MultiplayerSpaceShooter
