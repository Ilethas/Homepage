This project serves as a demo application that shows how to create various effects using OpenGL. It allows loading a 3D scene that is set up specifically for some effect. The user can then tweak various parameters, like for example exposure for HDR effect.

Every scene allows for its specialized operations, as well as basic ones, like:

  • Creating lights of different types (Point, Directional, Spot)
  • Modifying their properties (Position, Direction, Brightness, Attenuation Radius, Color, etc.)

Each scene has a customized "Description" tab that explains how the particular graphics effect works. A total of 8 OpenGL techniques are implemented:

  1. Lighting Models
  2. Normal Mapping
  3. Gamma Correction
  4. HDR
  5. Bloom
  6. Toon Outline Shader
  7. Gerstner Waves
