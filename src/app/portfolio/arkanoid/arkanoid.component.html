Arkanoid is my first game, that I made for a university project. The player controls a paddle with arrow keys (left & right) and can release the ball using the s key. The goal is to clear the playable area of all destructible blocks and not to let the ball fall below the paddle.

Sometimes the blocks drop a random reward upon destruction, which grants one of the following effects:

  • Sticky Paddle, the ball glues to the paddle upon contact. Can be released with the s key
  • Wide Paddle, makes the paddle wider
  • Shooting Paddle, makes it possible to shoot projectiles from the paddle using the f key.
  • Faster Balls, makes all balls faster
  • Slower Balls, makes all balls slower
  • Additional Balls, creates two additional balls to destroy blocks with

Slightly above the paddle, there are two spawn points for wisps, which change balls' velocity direction upon collision. The game features a scoreboard for the best players. It is possible to create new levels using notepad. The blocks correspond to letters A, B, C, D, E, X, while "." (the dot) marks an empty space.

The game can be downloaded from https://github.com/Ilethas/Arkanoid
